# DocOps
DocOps is a self-hostable Go API that handles  encrypted file upload, download, and search, routing to your own cloud storage provider.

Status: v0.1 in active development — 
local filesystem connector working, 
S3 coming in Sprint 5.

DocOps does not store your files — 
they go to your provider.

DocOps does not manage users or teams — 
it is a developer API, not an end-user product.

DocOps does not replace your storage provider — 
you still need S3, Google Drive, or similar.
